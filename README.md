# 🚀 Space Invaders — JAVAFatalPeroCompila

> Juego de Space Invaders hecho en Java para la asignatura de **Ingeniería del Software**.  
> Mata aliens, no mueras.

---

## ⚙️ Requisitos previos

Necesitas tener **Java 8 o superior** instalado. Para comprobarlo, abre una terminal y ejecuta:

```bash
java -version
```

Si el comando falla o no reconoce Java, descárgalo desde [java.com](https://www.java.com).

---

## ▶️ Cómo ejecutar el juego

**Opción 1 — La fácil:** haz doble clic sobre el archivo `.jar`.

**Opción 2 — Si nada ocurre:** abre una terminal en la carpeta donde esté el archivo y ejecuta:

```bash
java -jar SpaceInvaders.jar
```

## 🎯 Objetivo

Elimina **todos los aliens** antes de que lleguen al fondo de la pantalla.

- Si un alien **te toca** o **llega abajo** → 💀 pierdes.
- Si los **matas a todos** → 🏆 ganas.

Para empezar una partida, **haz clic sobre tu nave** en la pantalla de inicio.
